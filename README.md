# dbt Bootcamp Student Repository

This is a student repository for the [Complete dbt (Data Build Tool) Bootcamp: Zero to Hero](https://www.udemy.com/course/complete-dbt-data-build-tool-bootcamp-zero-to-hero-learn-dbt/) course on Udemy.

## Course Overview

The course covers dbt and Snowflake through theory and hands-on practice. The main project uses Airbnb data.

This repository contains my coursework, projects, and learning materials.
